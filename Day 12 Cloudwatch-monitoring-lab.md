# Day 12 – AWS Security Monitoring Lab

## Objective

Implement monitoring and alerting using AWS CloudWatch.

## Steps Performed

1. Logged into AWS console
2. Opened CloudWatch dashboard
3. Created monitoring alarm for EC2 CPU usage
4. Configured SNS email alert notifications

## Security Benefits

CloudWatch enables continuous monitoring of cloud resources and helps detect unusual activity or system issues.

Automated alerts allow rapid response to potential security incidents.

## Compliance Mapping

Control Type: Detective Control

ISO 27001 – Security Monitoring
NIST – Continuous Monitoring
SOC 2 – Security Event Monitoring
