# Day 1 – Cloud Security Governance

## What is Security Governance?

Security governance defines how an organization manages and controls its security program.

It includes policies, standards, procedures and security controls.

Security governance ensures security aligns with business objectives and regulatory requirements.

## Key Governance Elements

Policy
High level rules defined by leadership.

Standards
Mandatory technical requirements.

Procedures
Step-by-step implementation instructions.

Controls
Security safeguards protecting systems and data.

Compliance
Ensuring alignment with frameworks.

## Security Frameworks

ISO 27001
NIST
SOC 2
GDPR
DPDP Act 2023

## Example Governance Control

Firewall rule review process to ensure only authorized network traffic is permitted.
