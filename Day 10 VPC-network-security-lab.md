# Day 10 – AWS Network Security Lab

## Objective

Implement network security controls using AWS VPC Security Groups and Network ACLs.

## Steps Performed

1. Opened AWS VPC dashboard
2. Created security group (grc-web-security-group)
3. Configured inbound rule allowing HTTP (port 80)
4. Configured SSH access for administration
5. Reviewed Network ACL rules

## Security Benefits

Security Groups act as instance-level firewall controls.

Network ACLs provide subnet-level traffic filtering.

Together they enforce network segmentation and restrict unauthorized access.

## Compliance Mapping

Control Type: Preventive Control

ISO 27001 – Network Security Controls
NIST – Access Control
SOC 2 – Logical Access Security
