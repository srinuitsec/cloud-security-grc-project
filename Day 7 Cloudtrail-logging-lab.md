# Day 7 – Cloud Logging and Monitoring Lab

## Objective

Implement AWS CloudTrail logging to monitor cloud activity.

## Steps Performed

1. Logged into AWS console
2. Created CloudTrail trail (grc-security-trail)
3. Configured S3 bucket for log storage
4. Enabled logging for all regions

## Testing

Generated AWS activity by viewing EC2 instances.

Verified that events appeared in CloudTrail Event History.

## Security Benefit

CloudTrail enables monitoring of all AWS API activity and supports incident investigation.
