# Day 5 – Compliance Framework Mapping

## What is Compliance Mapping?

Compliance mapping links implemented security controls with compliance framework requirements.

It helps organizations demonstrate that security controls meet regulatory and compliance standards.

## Major Security Frameworks

ISO 27001
NIST
SOC 2
GDPR
DPDP Act 2023

## Example Security Control Mapping

| Security Control | Compliance Requirement |
|-----------------|-----------------------|
| Firewall rules | Network security control |
| IAM least privilege | Access control |
| Logging and monitoring | Security monitoring |

## Example Cloud Security Controls

IAM access restrictions – Access control requirement

Security groups – Network security requirement

CloudTrail logging – Monitoring requirement

Encryption using KMS – Data protection requirement
